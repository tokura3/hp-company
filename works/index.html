事例1：レポート自動化支援

領域：業務自動化 / 帳票出力
課題：定型レポートの作成・確認・配布に手作業が多い

対応内容

SQLによるデータ抽出、Pythonによる帳票生成、Google Driveへの保存処理を設計・実装。 手作業で行っていた集計・資料作成・配布業務の自動化を支援しました。

使用技術

  • SQL Server
  • Python
  • PDF出力
  • Google Drive

事例2：売上分析基盤構築

領域：データ基盤 / 売上分析
課題：売上データが複数システムに分散し、分析しづらい状態

対応内容

複数システムのデータを取得・加工し、分析しやすい形に整備。 BIツールでの可視化を前提に、集計用テーブルやデータマートの設計を支援しました。

使用技術

  • BigQuery
  • Snowflake
  • SQL
  • Tableau

事例3：BIダッシュボード導入支援

領域：BI / KPI可視化

事例4：SQLチューニング支援

  • データ抽出設計
